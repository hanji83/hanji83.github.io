Ruby is a dynamic, reflective, object-oriented, general-purpose programming language. Yukihiro Matsumoto (松本行弘) designed the language using C and released the first version on December 1995.

Rails, also known as Ruby on Rails, is an open source, full-stack framework for web applications. This framework emphasizes the use of software engineering patterns such as active record pattern, convention over configuration, don't repeat yourself, and model-view-controller. Rails was created by David Heinemeier Hansson. The first version was released on December 2005.

JavaScript is a computer programming language that follow multiple paradigms: object-oriented, imperative, and functional. It is a prototype-based scripting language with syntax influenced by C and names and naming convention from Java. JavaScript is primarily used for web browsers. The language was designed by Brendan Eich and first released as LiveScript in September 1995.

Backbone.js is a JavaScript library based on the model-view-presenter design paradigm. It has a dependency on another JavaScript library, Underscore.js. Backbone.js was designed for developing single-page web applications and synchronization of the web application between client and server. Backbone was created by Jeremy Ashkenas and first released in October 2010.

jQuery is a JavaScript library designed to simplify the client-side scripting of HTML. It allows for DOM elements selection, animation creation, event handling, and Ajax application development. jQuery was initially developed and released by John Resig on August 2006.

SQL, also known as Structured Query Language, is a multi-paradigm programming language designed for managing data stored in a relational database management system. SQL allows for data insert, query, update, delete, schema creation and modification, and access control. SQL was developed by Donald D. Chamberlin and Raymond F. Boyce. PostgreSQL is an object-relational database management system. It implements most of the SQL 2011 standard. Postgres was first released in May 1995.

CSS, also known as Cascading Style Sheets, is a language used for formatting a document written in markup language. It is often used for styling web pages and interfaces written in HTML.

Git is a distributed version control system software. It has version tracking capabilities, allows for non-linear development, distributed development, crytographic authentication of history, and many other features. Git was originally developed and designed by Linus Torvalds and initially released in April 2005.

C is a language that follows the imperative (procedural) and structured paradigms. It allows for low-level access to memory and provides language constructs that link well to machine instructions. C is a general-purpose, cross-platform language. It was initially developed by Dennis Ritchie in 1969.
